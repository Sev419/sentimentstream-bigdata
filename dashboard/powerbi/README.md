# Power BI Dashboard

Esta carpeta queda reservada para el dashboard principal del proyecto.

Fuente recomendada:

- API Flask: `GET /sentiments` y `GET /stats`
- MongoDB: base `sentimentstream_db`, colección `predictions`

Visualizaciones sugeridas:

- Conteo por sentimiento predicho.
- Últimas predicciones.
- Evolución por micro-batch si existe `microbatch_id`.
- Tabla de textos y predicciones.
